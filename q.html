The team at an Amazon warehouse is given a task to optimize the packing of a set of boxes with different IDs. Each box is labeled with an ID, and these boxes are currently arranged in a single row from left to right, where the id of the ith box is represented by the string s_id consisting of digits from 0 to 9 inclusive. To make the packing more efficient, the team can perform the following operation any (possibly zero) number of times:

  • Choose an index i and remove the digit s_id[i]. Then insert the box with ID min(s_id[i] + 1, 9) on any position (at the beginning, at the end, or in between any two adjacent boxes) in the row.

 

Given a string s_id, find the lexicographically minimal string of boxes using these operations.

 

Note: A string X is lexicographically smaller than a string Y of the same length if and only if in the first position where X and Y differ, the string X has a smaller digit than the corresponding digit in Y.

 

Example

Given, s_id = "26547".

 

Hence, the string returned is "24677". It can be proved that this is the lexicographically minimal string possible.

 

Function Description

Complete the function getMinimumString in the editor below.

getMinimumString has the following parameter:

    string s_id:  the ID of the boxes

 

Returns

    string: the lexicographically minimal string.

 

Constraints

  • 1 ≤ |s_id| ≤ 2*105 
  • s_id consists of only digits from 0 to 9. Note that s_id is just a string consisting of digits, so leading zeros are allowed.

 

Input Format For Custom Testing

The first and the only line which contains a string, s_id.

Explanation

The series of operations that shall be performed are:

  • Delete 8 and insert 9 at the end of the string. So the resulting string is "04299".
  • Delete 4 and insert 5 in the 3-rd position of the string. The resulting string is "02599".

Hence, the string returned is "02599". It can be proved that this is the lexicographically minimal string possible.

Explanation

Here, s_id = "34892".

The series of operations that shall be performed are:

  • Delete 3 and insert 4 at the end of the string. So the resulting string is "48924".
  • Delete 4 and insert 5 at the end of the string. So the resulting string is "89245".
  • Delete 8 and insert 9 at the end of the string. So the resulting string is "92459".
  • Delete 9 and insert 9 at the end of the string. So the resulting string is "24599".

Hence, the string returned is "24599". It can be proved that this is the lexicographically minimal string possible.

import java.util.ArrayList; import java.util.Collections; import java.util.List; public class MinimumString { public static String getMinimumString(String s_id) { int n = s_id.length(); char[] s = s_id.toCharArray(); char[] minSuffix = new char[n]; minSuffix[n - 1] = s[n - 1]; // Build a minimum suffix array—minSuffix[i] holds the minimum digit from s[i] to s[n-1] for (int i = n - 2; i >= 0; i--) { minSuffix[i] = (char) Math.min(s[i], minSuffix[i + 1]); } // List A: non-operated digits (kept in their original order) // List B: operated digits (digit incremented by 1, capped at 9) List nonOperated = new ArrayList<>(); List operated = new ArrayList<>(); for (int i = 0; i < n; i++) { // For the last digit no future digit is available so we cannot improve it—keep it. if (i == n - 1 || s[i] <= minSuffix[i + 1]) { nonOperated.add(s[i] - '0'); } else { // Mark digit for operation. (Operation: digit becomes min(digit+1,9)) int d = s[i] - '0'; operated.add(d == 9 ? 9 : d + 1); } } // The non-operated digits are in increasing order (this condition forces that) // Now sort operated digits. Collections.sort(operated); // Merge the two sorted lists to get the lexicographically minimal result. StringBuilder result = new StringBuilder(); int i = 0, j = 0; while (i < nonOperated.size() && j < operated.size()) { if (nonOperated.get(i) <= operated.get(j)) { result.append(nonOperated.get(i)); i++; } else { result.append(operated.get(j)); j++; } } while (i < nonOperated.size()) { result.append(nonOperated.get(i)); i++; } while (j < operated.size()) { result.append(operated.get(j)); j++; } return result.toString(); } public static void main(String[] args) { // Sample test cases: String s1 = "04829"; // Expected output: "02599" String s2 = "34892"; // Expected output: "24599" String s3 = "26547"; // Expected output: "24677" System.out.println("Result for \"" + s1 + "\": " + getMinimumString(s1)); System.out.println("Result for \"" + s2 + "\": " + getMinimumString(s2)); System.out.println("Result for \"" + s3 + "\": " + getMinimumString(s3)); } }
